Our Purpose

We want to tackle the problem of authorship regarding authentic/original choreography by dancers. Many social media platforms portray dances from creators without proper credibility shown to the main creator/author. We want to build a functionality that properly credits the author of their dance by creating a 3d mesh of the original choreography and detecting if another user uses their dance. If the dance is detected to not be original, we will automatically tag the author user on the video.
